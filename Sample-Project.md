
## Sample Project ##

- **Create directory and clone below repository**

  ```
  mkdir DockerD10

  cd DockerD10

  git clone https://github.com/kuldeepmehra27/DockerD10.git
  ```
- **Run following command to create container**
  ```
  docker compose up -d .
  ```
